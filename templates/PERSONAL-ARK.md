# PERSONAL ARK
## Emergency Identity & Recovery Document Template

---

> **What is this?**
> 
> The Personal Ark is a comprehensive template for documenting all critical personal 
> information needed to recover your identity, access your accounts, contact your 
> family, and resume life operations in any emergency scenario.
>
> Think of it as the "break glass in emergency" document for your entire life.

---

## Usage Instructions

1. **Copy this template** to a private location (never store filled version in git)
2. **Fill in your information** - be thorough, this is your lifeline
3. **Encrypt the filled version** using GPG, age, or VeraCrypt
4. **Store encrypted copies** in multiple locations:
   - Primary computer (encrypted)
   - USB backup drive (encrypted)
   - Cloud storage (encrypted before upload)
   - Paper copy in fireproof safe (for critical sections)
5. **Review quarterly** and update immediately when anything changes
6. **Tell trusted people** where to find this document if you're incapacitated

### Encryption Examples

```bash
# Using age (recommended - simple, modern)
age -p PERSONAL-ARK-FILLED.md > PERSONAL-ARK-FILLED.md.age

# Using GPG
gpg -c PERSONAL-ARK-FILLED.md

# Decrypt
age -d PERSONAL-ARK-FILLED.md.age > PERSONAL-ARK-FILLED.md
gpg -d PERSONAL-ARK-FILLED.md.gpg > PERSONAL-ARK-FILLED.md
```

### NEVER

- Store the filled version unencrypted
- Commit the filled version to git
- Upload the filled version to cloud without encryption
- Email the filled version
- Store actual passwords in this document (use hints only)
