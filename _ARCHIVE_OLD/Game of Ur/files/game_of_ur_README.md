1. Open `royal_game_of_ur.html` in any web browser
2. Choose **Solo Divination** (oracle reading) or **Two Players** (Dawn vs Dusk)

## The Oracle

Each bird governs a domain of fortune. As you play, the Oracle Log builds your reading:

- **Land on a rosette**: Positive fortune
- **Pass over a rosette**: Negative fortune
- **Complete the journey**: Your bird's domain is fulfilled

The pattern of fortunes received‚Äîwhich birds entered, which struggled, which landed true‚Äîforms the message from fate.
