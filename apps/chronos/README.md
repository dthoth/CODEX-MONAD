# Chronos

**Version 1.0.0**

The Count & The Counting - Countdown to upcoming horizons

---

Chronos is part of the CODEX-MONAD ecosystem. A CRT-styled countdown clock for tracking events and deadlines with amber-phosphor aesthetics.

## Features

- **Hero Display** - Large countdown to the nearest upcoming event
- **Event Management** - Add, edit, and delete countdown events
- **Persistent Storage** - Events saved to localStorage
- **Import/Export** - JSON backup and restore
- **Past Events** - Toggle visibility of completed countdowns
- **CRT Aesthetics** - Scanlines, vignette, and amber glow

## Usage

1. Open `index.html` in your browser
2. Add events with name, icon, and target date/time
3. Watch the countdown tick
4. Export events for backup
