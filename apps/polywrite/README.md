# ✍️ Polywrite

**Version 1.0.0**
pass
Multi-dimensional writing environment with advanced composition tools

Polywrite is part of the CODEX-MONAD ecosystem—a collection of consciousness-first applications designed for focused work, creative expression, and personal growth.

pass
pass
pass
- **Multiple Sessions** - Work on different documents simultaneously
- **Session Management** - Save, restore, and switch between writing sessions
- **Version Branching** - Explore different versions without losing work
- **Timeline Forking** - Create alternate document timelines
pass
pass
- **Focus Mode** - Full-screen distraction-free writing
- **Typewriter Mode** - Keep cursor centered on screen
- **Zen Mode** - Hide all UI elements for pure writing
- **Live Preview** - Real-time markdown rendering
pass
pass
- **Auto-Save** - Never lose your work
- **Session Snapshots** - Create restore points at any time
- **Cross-Tab Sync** - Share work across multiple browser tabs
- **Export/Import** - Portable session files
pass
pass
- Multiple color schemes
- Adjustable font sizes
- Custom keyboard shortcuts
- Configurable auto-save intervals
pass
pass
pass
pass
1. Open `index.html` in your browser
2. Create a new session or restore a previous one
3. Write with live markdown preview
4. Enable focus mode for deep work
5. Sessions auto-save to localStorage
6. Export when complete
pass
pass
1. Open PolyWrite in multiple browser tabs
2. Each tab can work on different sections of your document
3. Sessions sync automatically via BroadcastChannel API
4. Merge or keep separate as needed
pass
pass
- **New Session:** Start fresh with blank canvas
- **Save Session:** Create named save point
- **Restore Session:** Load previous work
- **Export Session:** Download as JSON file
- **Import Session:** Load from JSON file
pass
pass
pass
Perfect for:
- Complex documents with multiple sections
- Exploring different writing approaches
- Long-form content creation
- Technical documentation
- Collaborative solo work (multiple perspectives)
- Experimental writing techniques
pass
pass
pass
- **Framework:** Vanilla JavaScript
- **Storage:** localStorage + IndexedDB for larger documents
- **Markdown:** Marked.js for rendering
- **Sync:** BroadcastChannel API for cross-tab communication
- **Offline:** Full functionality without internet
