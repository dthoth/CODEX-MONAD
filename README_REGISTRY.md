## Troubleshooting

**Apps don't appear:**
- Check browser console (F12)
- Verify codex-apps.json exists
- Use HTTP server (not file://)

**Generator fails:**
- Check Python version (need 3.6+)
- Verify apps/ directory exists
- Check app.json files are valid JSON

**Apps can't launch:**
- Verify paths in codex-apps.json
- Check files exist at hubPath locations
